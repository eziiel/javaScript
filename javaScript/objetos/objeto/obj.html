Object.defineProperties(alvo, propiedades)
com esse comando da para colocar propiedades nos objetos de outro modo, no qual essa propiedade poderá ser imutável, ou mutável caso desejado, pode receber dois argumetos pós definidas as propiedades: precida do value entre chaves, configurable, enumerable, writable
ex linha 41
Object.defineProperties(objetoParaColocarAsPropiedades, {
valor1: {
value : 'valorAtribuido',
}
configurable: true, //opcional
writable: true//opcional
enumerable: true,//opcional
})
