Array

armazena uma coleção de elementos

  • array
  • function
  • objetos
  • boolean

Dois modos de fazer

  1. direto: const num = [1, 2, 3]
  2. indireto: const num = new Array(1,2,3)

Metodos de Array

Array.from()
metodo para transformar array-like object em array.
arr.join(parametro)
retorna os valores de um array em string, retornando o parametro como separador, se nao for definido, por padrao retorna vírgula
arr.reverse()
reverte a ordem da array
arr.concat()
concatenar arrays
arr.slice(parametro1, parametro2)
vale o mesmo que para strings, retorna o valor de uma array conforme os parametros passados, podendo ser omitodo o inicio e o final pode receber valor negativo

pode usar o slice para clonar uma array: arr.slice()

arr.splice(parametro1, parametro2, parametro3)
quase igual slice, remove items da array conforme os parametros passados(a partir do parametro1 remove(quantidade)parametro2), a partir do 3 é tudo adcionado
array.pop() - array.push()
pop remove no final e push adiciona no final, podem ser vazios os parametros, mas pelo menos no push o ideal é colocar um param
arr.unshift() e arr.shift()
remove ou adiciona no inicio de cada arraypodem ser vazios os parametros, mas pelo menos no push o ideal é colocar um param
arr.toString()
transforma a array em string
arr.copyWithing(alvo, inicio, final)
faz uma copia de determinados elementos da array na propia array
arr.fill(parametros, inicio, final)
preenche a array com o parametro passado, o inicio e final podem ser opcional
arr.includes(paramentro)
verifica na array se o parametro passado esta na array
arr.indexOf(parametro)
retorna a posição do primeiro paramentro no array mesmo tendo mais. ps:começa do 0
arr.lastIndexOf(paramentro)
retorna a posição do primeiro paramentro no array começando do final mesmo tendo mais. ps:começa do 0
arr.sort()
classifica os items da array, se não colocar parametro fica organizado por ordem alfabetica

Metodos de Array pelo ecmas5

ECMAScript 5 define nove novos métodos de array para iterar, mapear, filtrar, testar, reduzir e pesquisar arrays.

a maioria dos elementos aceita uma funcao como primeiro argumento e chama essa funcao para cada elemento, se o array for esoparso a funcao nao pe passada para elements inexistentes.

na maioria dos casos a funcao passada recebe tres parametros, item, indice, array

arr.forEach((item, index, array) function {})
forEach é como loop, interando para cada elemento, pode receber tres parametros, e aplicar uma funcao para cada um, essa funcao pode ser arrow, o importante no forEach é o que faz com cada elemento e não o retorno de sua inicialização

ps: forEach nao pode ser parado normalmente com o break

***PS: Tanto forEach() e MAP(), podem receber arrow function, podem omitir os parenteses() do parametro se for somente um e omitir as chaves se a linha de código for somente uma, e ambas poder recerber uma callback function de fora

****map() pode omitir o return se for de uma linha só

arr.map((item, index, array)function {})
se o array for esparso o metodo map() retorna um resultado esparso, é parecida com forEach(), porem a map() retorna um valor, um novo array não modificando assim o que foi chamado, e assim precisa de um return

map é uma funcao de transformacao

arr.filter((item, index, array)function {})
espera uma callback de valor booleando e ele passa pela array retornando todos os items que forem verdadeiros (linha 33)
arr.filterIndex((item, index, array)function {})
espera uma callback de valor booleando e ele passa pela array retornando o index que for verdadeiro
arr.reduce(callback{}(acumulador, item, index) ,valor inicial)
executa uma funcao de callback para cada item, reduce recebe dois parametros novos, um acumulador que acumula os dados e é retornado, e um valor inicial para o acumulador.

o acumulador é o retorno da interação anterior

arr.reduceRight()
reduce mas começando da direita
arr.some()
- não é pra somar. tem como objetivo retornar um booleano, retorna verdadeiro se tiver pelo menos um verdadeiro, e Falso se todos forem falsos
arr.every()
retorna falso se um for false, e só retorna verdadeiro se todos forem verdadeiros
arr.find()
ele retorna o primeiro item (valor) verdadeiro (linha 31)
arr.findIndex()
igual o find, porem retorna o index
